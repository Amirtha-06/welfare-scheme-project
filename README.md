# Welfare Intelligence Platform

A hybrid Full-Stack and Data Science web application designed to administer, manage, and analyze various welfare schemes. 

## Features
- **Data Science Integration**: Leverage data analytics for scheme insights and intelligent decision-making.
- **User Registration & Authentication**: Secure access for administrators and users.
- **Admin Dashboard**: Centralized view for managing welfare programs and viewing analytics.
- **Scheme Management**: Add, edit, and safely manage comprehensive welfare schemes.
- **Data Persistence**: Uses a local SQLite database and CSV files for robust data management and analysis.

## Tech Stack
- **Full Stack Development**: Python, Flask, HTML, CSS, Jinja2 Templates
- **Data Science**: Python Data Ecosystem (e.g., Pandas, Scikit-learn, etc.)
- **Database**: SQLite

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/welfare_scheme_project.git
   cd welfare_scheme_project
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```
3. Install dependencies (assuming you have a requirements.txt):
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
